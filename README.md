# ForecastIQ

AI-Powered Revenue & ROAS Intelligence Platform

## Problem

Digital marketers struggle to forecast future revenue, estimate ROAS, optimize budget allocation, and quantify uncertainty across advertising channels.

## Solution

ForecastIQ combines machine learning, probabilistic forecasting, budget simulation, and marketing intelligence to provide actionable campaign insights.

## Features

* Revenue Forecasting
* ROAS Forecasting
* Campaign-Level Forecasts
* Campaign Type Forecasts
* Budget Scenario Simulator
* Monte Carlo Forecasting (P10/P50/P90)
* Risk Detection Center
* Budget Optimization Engine
* AI Recommendations
* Executive Summary Dashboard

## Tech Stack

* Python
* XGBoost
* Scikit-Learn
* Pandas
* Streamlit
* Plotly
* Monte Carlo Simulation

## Dashboard

ForecastIQ enables marketers to:

* Forecast future revenue
* Forecast ROAS
* Simulate budget scenarios
* Detect campaign risks
* Optimize budget allocation
* Quantify forecast uncertainty
